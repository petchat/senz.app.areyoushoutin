** Setup Eclipse project **
For a correct import of this project you need to:
- Open Eclipse
- Select File->Import
- Select "Existing Projects into Workspace" from tab "General" and click "next".
- Select archive file and Browse to "numberpicker-lib_Eclipse_archive.zip"
- Click "Finish"
- Redo previous steps and import "AreYouShoutingX.X_Eclipse_archive.zip" file
- Now the project has been added to your Workspace!
